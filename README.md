# CSES Code Golf Solutions

Accepted [code golf](https://en.wikipedia.org/wiki/Code_golf) solutions to the [CSES Problem Set](https://cses.fi/problemset/) written in Python 3. Code golf solutions have file names ending in `Golf` while my standard solutions do not. After solving a CSES problem, you can click `QUEUE` and concatenate `?lang=9&status=2&user=&by=2&order=0` to the URL to show the Python 3 shortest code leaderboard.

For problems labeled with a üêç, I am #1 on the shortest code leaderboard for Python 3 after disregarding users who execute whitespace language.

For problems labeled with a ‚≠ê, I am #1 on the shortest code leaderboard out of ALL programming languages.

For problems labeled with a üí•, I have the first accepted solution in Python 3.

For problems labeled with a üåä, I have also golfed the problem in C++.

For problems labeled with a üóø, the number of non-whitespace characters cannot be shortened for Python 3.
